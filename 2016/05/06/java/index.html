Mac配置maven环境

1、首先去下载maven https://maven.apache.org/download.cgi

2、把maven放到一个位置。例如：/Users/wangshuaiguang/Desktop/java/maven/apache-maven-3.3.9 目录下

3、打开命令行：

4、eclipse配置maven 为eclipse建立独立Repository
新建一个文件夹作为repository的根目录， 将setting文件加入

之后在eclipse中安装m2eclipse插件（在marketplace可找到）

然后在eclipse的偏好设置里设置maven的installations 选择之前解压出来的maven文件路径

最后，进入User Setting 设置setting.xml文件路径
