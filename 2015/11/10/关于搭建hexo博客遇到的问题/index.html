第一篇文章 关于搭建hexo博客遇到的问题

标签（空格分隔）： Mac iOS hexo

1、安装Node.JS 安装git
2、上两步完成之后，安装通过npm 安装hexo
sudo npm install -g hero 有的不需要加sudo
3、初始化一个hexo
hexo init
4、安装hexo
npm install
5、创建hexo
hexo generate
6、启动服务器
hexo server

现在访问 localhost：4000就可以在本地的浏览器打开hexo

链接github
1、在Github上建立repository
非常要注意的是：建立的名字应该是你的github账户名.github.io 固定的格式

2、修改_config.yml里的deploy 文件里的type
repository输入为https：需要在命令行中输入用户名和密码；
repository输入为ssh ：需要在github上添加ssh
deploy:
type: git
repository: https://github.com/Sunny-guangge/Sunny-guangge.github.io.git

branch: master
3、修改完成之后，把代码同步到git上面
4、保存在_config.yml里的修改
npm install hexo-deployer-git –save
5、然后创建
hexo generate
6、然后同步到服务器上
hexo deploy
7、如果是https 则需要输入用户名 然后输入github 账号密码

然后访问 github账户名.github.io 就可以访问了
