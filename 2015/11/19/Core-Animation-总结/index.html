Core Animation 总结

CABasicAnimation CAKeyframeAnimation CATransition

CALayer的子类 CAReplicatorLayer

1、 CAReplicatorLayer创建layer和它的sublayer的多个副本，副本可以设置transform来变形，或者设置颜色、透明度的变化

2、雪花的飘动，燃烧的火焰，烟雾效果，其实都是通过这个类的设置来实现的 CAEmitterLayer 和 CAEmitterCell

3、 CAGradientLayer用来绘制渐变色，指定几个颜色值、渐变结束位置，就能在layer中绘制出渐变效果

4、更多的layer子类：CATransformLayer 、 CATiledLayer 、 CAScrollLayer
